Сюда кладёшь картинки для карточек слов/ролей и шпиона.

Полный список нужных файлов (с точными именами) — в `IMAGES_NEEDED.md`
в корне проекта.

Формат имени файла: `<слаг>.png` (или .jpg/.jpeg/.webp) — слаг это слово
в нижнем регистре с подчёркиваниями вместо пробелов, без знаков
пунктуации. Пример: слово "Анти-Маг" → файл `анти_маг.png`.

Если для какого-то слова файла тут не найдётся — бот сам подставит
простую текстовую карточку вместо картинки, игра не сломается.
